# 🧮 Calculator Web App

> ⚡ Một ứng dụng máy tính đơn giản chạy trực tiếp trên trình duyệt.

🌐 **Live Demo:** https://panbap.github.io/Calculator/

---

## 🚀 Giới thiệu

Đây là một project **Calculator (máy tính)** được xây dựng bằng JavaScript thuần, giúp thực hiện các phép toán cơ bản một cách nhanh chóng và trực quan.

Phù hợp để:
- 🧑‍💻 Luyện tập JavaScript cơ bản  
- 📚 Hiểu cách xử lý sự kiện (event handling)  
- 🎯 Demo mini project front-end  

---

## ✨ Tính năng

- ➕ Cộng, ➖ Trừ, ✖️ Nhân, ➗ Chia  
- 🧼 Nút xóa (AC / DEL)  
- 📱 Giao diện thân thiện, dễ dùng  
- ⚡ Xử lý nhanh, không cần backend  

---

- JavaScript
